Structure des Dossiers
Les deux dossiers que vous trouverez contiennent les éléments suivants :

Répertoire src : Contient le code source commenté pour faciliter la compréhension du programme.
Répertoire doc : Contient la Javadoc, qui explique en détail le fonctionnement des classes et méthodes utilisées.
Répertoire dist : Contient l'exécutable ou la librairie du projet, prêt à être utilisé.
Fichier build.xml : Ce fichier est repris du cours pour la gestion de la construction du projet via Apache Ant. Pour que le projet fonctionne correctement, il est nécessaire de créer des répertoires supplémentaires, comme le répertoire test.
Un rapport : Chaque dossier contient un rapport différent, détaillant le travail effectué et les étapes du projet.
Informations supplémentaires :
Les rapports sont mis en avant sur GitHub pour une meilleure accessibilité.
